# DWP Polije Mobile

DWP POLIJE Mobile adalah Sebuah aplikasi berbasis mobile untuk manajemen dan monitoring bazar pada salah satu organisasi yang berada di Politeknik Negeri Jember yaitu Dharma Wanita Persatuan.
